Initialization command

This edition does not provide by default different files for different external tools. But a command is available to load and have such files.

{% highlight bash %} $ php app/console bigbang:init [--behat] [--ant|--jenkins] {% endhighlight %}
For the moment only the following options are supported :
  • --behat : will add a behat.yml in your root directory and add a features directory with a base Context class.
  • --ant or --jenkins (alias) : will add a build.xml in your root directory that contains a base set of ant tasks for code quality and continous integration.
