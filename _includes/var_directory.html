Var directory

BigBang edition changes the standard way to store cache and log files. They are in the var directory, at the root of your project, and follow this architecture :

{% highlight php %} +- app +- src +- var +- cache | +- dev | +- prod | +- test +- logs +- sessions +- vendor {% endhighlight %}

You can see that the PHP sessions files are also stored in the var directory (and not in the cache directory contrary to the standard edition). That allow to keep sessions even if the cache is cleared.
