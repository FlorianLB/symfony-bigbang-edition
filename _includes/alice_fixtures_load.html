Command for loading alice fixtures

{% highlight bash %} php app/console alice:fixtures:load dir --{append|truncate-schema|drop-schema} {% endhighlight %}
Your fixtures files must be placed in your bundle in Resources/fixtures/your-dir : {% highlight php %} +- FooBundle | +- Resources | | +- fixtures | | | +- prod | | | +- test | | | | +- User.yml | | | | +- Product.yml +- BarBundle +- PonyBundle {% endhighlight %}
If you want to load all the test fixtures of the test FooBundle (with a truncate schema strategy) : {% highlight php %} php app/console alice:fixtures:load test --truncate-schema --bundles=FooBundle {% endhighlight %}

--bundles or -b option allow filtering on bundles name. You can give a full bundle name or just a pattern.

You must choose between 3 strategy :
  • drop-schema : it will drop and create the schema before loading the fixtures
  • truncate-schema : it will truncate all the tables of schema before loading the fixtures (faster than drop)
  • append : simply add the fixtures
